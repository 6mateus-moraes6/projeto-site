Curso com Git e GitHub

Git é um sistema de controle de versão distribuído que permite rastrear alterações em arquivos. GitHub é uma plataforma de hospedagem de código-fonte baseada no Git que permite compartilhar e colaborar em projetos.

Hospedagem de Site com GitHub

A hospedagem de sites com o GitHub permite que você aloje páginas da web diretamente de repositórios Git. Ao criar um repositório com o nome de usuário seguido por ".github.io", o GitHub automaticamente converte o conteúdo em uma página estática acessível pela web.
